Auto Whitespace Formatting
=================
![Chrome Web Store Version](https://img.shields.io/chrome-web-store/v/eadgahnmkiekfdidigefdpmmankejndk)

A small browser extension to clean up extra whitespace.

Features
- Remove leading/trailing whitespace from fields (except password)
- Collapse multiple line breaks
- UI with configurable settings
- 3 Modes - Allow list, Deny List, All Sites (default)

Usage
- Click the extension icon to open the popup, then use the provided controls to configure how you like.
